# GRootR 0.0.0.9000

## New Features

- Initial release of **GRootR**, a package developed for the visualization and analysis of tree root systems detected using ground-penetrating radar (GPR) with 3D scanning capabilities.
- Core features of the package include:
  - Importing and processing GPR data in CSV format.
  - Generating interactive 3D visualizations of root systems in the soil.
  - Performing statistical analyses of root distribution, density, and spatial variability.

## Project Context

This package is part of the research project titled:
**"Zastosowanie georadaru ze skanerem 3D do pomiarów zmienności systemów korzeniowych sosny zwyczajnej"**  
(**"Application of Ground-Penetrating Radar with 3D Scanner for Measuring Variability in Scots Pine Root Systems"**)

- **Project duration**: 11.07.2024 - 10.07.2025  
- **Project number**: G-1447/WL/24-25  
- **Funding agency**: Narodowe Centrum Nauki (National Science Centre, Poland)  
- **Principal investigator**: Dr inż. Luiza Tymińska-Czabańska
